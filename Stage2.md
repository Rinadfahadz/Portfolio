# 📘 Project Charter

- **Project Name:** Hackathon Team Formation Platform  
- **Stage:** Stage 2 – Project Charter

- **Date:** March 2026  

---

## 🎯 1. Project Objectives

### Purpose
The purpose of this project is to develop a web-based platform that helps hackathon participants form teams efficiently based on skills, roles, and interests, while ensuring trust through an idea protection system.

### Objectives
- Develop a functional MVP within 8 weeks including user profiles, team creation, and matching system  
- Reduce team formation time to under 10 minutes  
- Implement a basic idea protection system (timestamping and conversation tracking)

## 📦 3. Project Scope

### ✅ In Scope
- User registration and profiles  
- Team creation and management  
- Browse users and teams  
- Join request system  
- Basic matching system  
- Idea protection (timestamp + chat tracking)  

### ❌ Out of Scope
- Advanced AI matching  
- Full real-time chat  
- Mobile app  
- External integrations (GitHub, etc.)  
- Rating system  

---

## ⚠️ 4. Risks and Mitigation

| Risk | Mitigation |
|------|-----------|
| Limited time | Focus on MVP features only |
| Technical challenges | Start simple, improve later |
| Team miscommunication | Weekly meetings + clear tasks |
| Skill gaps | Allocate time for learning |
| Incomplete features | Prioritize core functionality |
