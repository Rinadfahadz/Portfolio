User Stories – Hackathon Teaming Platform

Persona 1: Participant (Individual Looking for a Team)

Must Have (MVP)

As a participant, I want to create a personal profile with my skills (e.g., programming, design), so that teams can evaluate me.
As a participant, I want to browse available teams looking for members, so that I can join a suitable team.
As a participant, I want to request to join a team, so that I can participate in a hackathon collaboratively.
As a participant, I want to view team details (project idea, required skills, members), so that I can decide if it fits me.
As a participant, I want to receive updates on my join requests (pending, accepted, rejected), so that I know my status.

Should Have

As a participant, I want to add additional info (portfolio, GitHub, experience), so that I increase my chances of being accepted.
As a participant, I want to view my past teams and hackathons, so that I can track my experience.

Could Have

As a participant, I want to rate my team experience, so that platform quality improves.

⸻

Persona 2: Team Leader (Team Creator)

Must Have (MVP)

As a team leader, I want to create a team with a project idea, so that I can attract members.
As a team leader, I want to specify required roles (e.g., frontend, backend, UI/UX), so that I get suitable applicants.
As a team leader, I want to receive join requests from participants, so that I can build my team.
As a team leader, I want to accept or reject applicants, so that I control team composition.
As a team leader, I want to view participant profiles, so that I can evaluate their skills.

Should Have

As a team leader, I want to manage my team members, so that I can update or remove members if needed.
As a team leader, I want to view my past teams and projects, so that I track my history.

Could Have

As a team leader, I want to rate team members after the hackathon, so that collaboration quality improves.

⸻

Persona 3: Hackathon Organizer

Must Have (MVP)

As an organizer, I want to create and publish hackathon events, so that participants can join.
As an organizer, I want to view all teams participating in my hackathon, so that I can manage the event.
As an organizer, I want to monitor registrations (participants and teams), so that the event runs smoothly.

Should Have

As an organizer, I want to provide hackathon details (rules, timeline, prizes), so that participants are informed.

Could Have

As an organizer, I want to highlight featured teams or ideas, so that engagement increases.

⸻

Persona 4: Admin (Platform Manager)

Must Have (MVP)

As an admin, I want to manage users (participants and team leaders), so that the platform stays safe.
As an admin, I want to monitor teams and hackathons, so that everything operates properly.
As an admin, I want to handle reports or issues, so that user trust is maintained.

⸻

Won’t Have (Out of MVP Scope)

The platform will not include real-time chat between users (initially).
The platform will not include video calls or meetings.
The platform will not automatically assign teams using AI.
The platform will not manage hackathon judging or submissions.
